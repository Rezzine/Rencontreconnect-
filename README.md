# RencontreConnect

**RencontreConnect** est un site de rencontre pour tout le monde, basé sur le respect.

## Description
Notre objectif est de créer un espace bienveillant où chacun peut faire des rencontres sincères, qu’elles soient amicales ou amoureuses.

## Fonctionnalités
- Accessible à toutes et tous
- Respect et sécurité au cœur de la plateforme
- Interface simple et fluide
- Développé par Rezzine

## À venir

- Système de modération communautaire
